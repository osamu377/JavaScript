### border-collapse: collapse; は「<ruby>魔法<rt>まほう</rt></ruby>の<ruby>言葉<rt>ことば</rt></ruby>」

```css
    width: 100%;

    padding: 12px 15px;
}
th {
    background-color: #4CAF50; /* ヘッダーは緑 */

}
tr:nth-child(even) {
    background-color: #f2f2f2;
}
```

>[!note]  
><ruby>注意<rt>ちゅうい</rt></ruby>：border（<ruby>枠線<rt>わくせん</rt></ruby>）は<ruby>各<rt>かく</rt></ruby>セル（`<th>`や`<td>`）に<ruby>設定<rt>せってい</rt></ruby>し、border-collapseは<ruby>全体<rt>ぜんたい</rt></ruby>の<ruby>親<rt>おや</rt></ruby>である`<table>`に<ruby>設定<rt>せってい</rt></ruby>するのが<ruby>正<rt>ただ</rt></ruby>しいルールです。

## III. <ruby>特定<rt>とくてい</rt></ruby>の<ruby>要素<rt>ようそ</rt></ruby>（クラス）のスタイル

JavaScriptには、`getElementsByClassName()` という「クラス<ruby>名<rt>めい</rt></ruby>を<ruby>使<rt>つか</rt></ruby>って<ruby>要素<rt>ようそ</rt></ruby>を<ruby>見<rt>み</rt></ruby>つける」<ruby>命令<rt>めいれい</rt></ruby>があります。CSSでクラスを<ruby>使<rt>つか</rt></ruby>ってグループ<ruby>化<rt>か</rt></ruby>しておけば、<ruby>後<rt>あと</rt></ruby>でJavaScriptからそのグループだけを<ruby>一度<rt>いちど</rt></ruby>に<ruby>動<rt>うご</rt></ruby>かすことが<ruby>非常<rt>ひじょう</rt></ruby>に<ruby>簡単<rt>かんたん</rt></ruby>になります。

## V. JavaScriptからCSSを<ruby>操作<rt>そうさ</rt></ruby>する<ruby>重要<rt>じゅうよう</rt></ruby>ルール

| CSSの書き方          | JavaScriptでの書き方 | 意味                                                           |
| ---------------- | --------------- | ------------------------------------------------------------ |
| font-size        | fontSize        | <ruby>文字<rt>もじ</rt></ruby>の<ruby>大<rt>おお</rt></ruby>きさ       |
| background-color | backgroundColor | <ruby>背景<rt>はいけい</rt></ruby>の<ruby>色<rt>いろ</rt></ruby>       |
| border-radius    | borderRadius    | <ruby>角<rt>かど</rt></ruby>の<ruby>丸<rt>まる</rt></ruby>み（かどのまるみ） |

**JavaScriptでの<ruby>書<rt>か</rt></ruby>き<ruby>換<rt>か</rt></ruby>え<ruby>構造<rt>こうぞう</rt></ruby>：「<ruby>道<rt>みち</rt></ruby>のり」で<ruby>考<rt>かんが</rt></ruby>える**

JavaScriptからCSSを<ruby>操作<rt>そうさ</rt></ruby>するときは、<ruby>次<rt>つぎ</rt></ruby>のような<ruby>構造<rt>こうぞう</rt></ruby>で<ruby>命令<rt>めいれい</rt></ruby>を<ruby>出<rt>だ</rt></ruby>します。


1. document：このウェブページという「<ruby>大<rt>おお</rt></ruby>きな<ruby>書類<rt>しょるい</rt></ruby>」の<ruby>中<rt>なか</rt></ruby>から、
2. .getElementById("title")：IDが "title" の「<ruby>場所<rt>ばしょ</rt></ruby>」を<ruby>見<rt>み</rt></ruby>つけて、
3. .style：「デザイン（スタイル）」の<ruby>担当者<rt>たんとうしゃ</rt></ruby>に、
