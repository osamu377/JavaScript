# JavaScriptとオブジェクト<ruby>指向<rt>しこう</rt></ruby>

プログラミングについて、いろいろ調べたことのある方なら、 **「オブジェクト<ruby>指向<rt>しこう</rt></ruby>プログラミング（Object Oriented Programming <ruby>略<rt>りゃく</rt></ruby>して"OOP"）」** という言葉を見たことがあるかもしれません。

OOPにはいろんな特徴がありますが、ここではそのうちの二（ふた）つを取り上げます。

1. いろいろなデータや<ruby>動<rt>うご</rt></ruby>きをオブジェクトとしてまとめ、整理することで、複雑（ふくざつ）なアプリを作りやすくする
2. すでにあるオブジェクトを再利用（さいりよう）して、それをもとに別のオブジェクトを作ることで、プログラミングの手間を省く

この章では 上の「1.」に関連して、**自分だけのオブジェクトを作る方法** と **すでにあるオブジェクトを使いやすいように改造する方法** を、説明し、「2.」に関連して、**クラスと<ruby>継承<rt>けいしょう</rt></ruby>** について説明します。
