# <ruby>入出力<rt>にゅうしゅつりょく</rt></ruby>

コンピューターの世界で最も大切な基本、**「<ruby>入力<rt>にゅうりょく</rt></ruby>（Input）」** と **「<ruby>出力<rt>しゅつりょく</rt></ruby>（Output）」** について説明します。

留学生の方には、コンピューターを **「<ruby>自動販売機<rt>じどうはんばいき</rt></ruby>」** や **「<ruby>料理<rt>りょうり</rt></ruby>」** に<ruby>例<rt>たと</rt></ruby>えて教えてあげると、とても分かりやすくなります。

***

## I. <ruby>入力<rt>にゅうりょく</rt></ruby>（Input）とは？

<ruby>入力<rt>にゅうりょく</rt></ruby>は、**「人間がコンピューターに情報（じょうほう）を与えること」** です。
コンピューターに「これをやって！」とお願いするステップです。

* **キーボード：** 文字を<ruby>入力<rt>にゅうりょく</rt></ruby>する。
* **マウス：** クリックして指示（しじ）を出す。
* **マイク：** 声を音のデータとして入れる。
* **カメラ：** 顔や景色を<ruby>画像<rt>がぞう</rt></ruby>として入れる。

***

## II. <ruby>出力<rt>しゅつりょく</rt></ruby>（Output）とは？

<ruby>出力<rt>しゅつりょく</rt></ruby>は、**「コンピューターが人間に<ruby>結果<rt>けっか</rt></ruby>を返すこと」**です。
コンピューターが頑張って計算したあとの「答え」を見せるステップです。

* **モニター（<ruby>画面<rt>がめん</rt></ruby>）：** 文字や絵を<ruby>表示<rt>ひょうじ</rt></ruby>する。
* **スピーカー：** 音を出す。
* **プリンター：** 紙に印刷（いんさつ）する。

***

## III. 分かりやすい<ruby>例<rt>たと</rt></ruby>え：自動販売機（Vending Machine）

コンピューターの<ruby>動<rt>うご</rt></ruby>きを、自動販売機で考えてみましょう。

1. **<ruby>入力<rt>にゅうりょく</rt></ruby> (Input):** あなたがお金を入れて、ボタンを押します。（指示を与える）
2. **処理 (Process):** 機械が「お金は足りるかな？」と計算します。（中での仕事）
3. **<ruby>出力<rt>しゅつりょく</rt></ruby> (Output):** ジュースが出てきます！（結果が返ってくる）

***

## IV. 留学生へのアドバイス：プログラムでの<ruby>書<rt>か</rt></ruby>き<ruby>方<rt>かた</rt></ruby>

JavaScriptなどのプログラミングを学ぶとき、最初に出会う「<ruby>入力<rt>にゅうりょく</rt></ruby>と<ruby>出力<rt>しゅつりょく</rt></ruby>」も教えてあげましょう。

* **<ruby>入力<rt>にゅうりょく</rt></ruby>の代表：`prompt()`**
ユーザーに文字を書いてもらうための<ruby>入力<rt>にゅうりょく</rt></ruby><ruby>窓<rt>まど</rt></ruby>を出します。
* **<ruby>出力<rt>しゅつりょく</rt></ruby>の代表：`console.log()`**
プログラムの結果をコンソール<ruby>画面<rt>がめん</rt></ruby>に<ruby>表示<rt>ひょうじ</rt></ruby>します。

```javascript
// 入力（にゅうりょく）（Input）
const name = prompt("あなたのお名前は？");

// 出力（しゅつりょく）（Output）
console.log("こんにちは、" + name + "さん！");

```

> [!TIP]
> **「I/O（アイオー）」ってなに？**
> ITの世界では、InputとOutputを合わせて **「I/O」** と呼ぶことが多いです。「データのやり取り」のことだと思っておけば大丈夫だよ！と伝えてあげてください。

いかがでしょうか？「入れる」と「出す」の関係がわかれば、プログラミングの<ruby>仕組<rt>しく</rt></ruby>みがぐっと理解しやすくなります。
