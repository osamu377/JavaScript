# <ruby>組<rt>く</rt></ruby>み<ruby>込<rt>こ</rt></ruby>み（くみこみ）オブジェクト

**オブジェクトとは**

まず「オブジェクト」について復習しましょう。一言でいうと、オブジェクトは **「関係（かんけい）がある 情報を まとめたもの」** でしたね。

そして、オブジェクトのすごいところは、**データ（特徴）** だけでなく、**<ruby>関数<rt>かんすう</rt></ruby>（<ruby>動<rt>うご</rt></ruby>き）** も一緒にまとめられることでした。

* **プロパティ（特徴）：** 名前、身長、色、値段など
* **メソッド（<ruby>動<rt>うご</rt></ruby>き）：** 走る、話す、計算するなど

**まとめ**

JavaScriptの<ruby>組<rt>く</rt></ruby>み<ruby>込<rt>こ</rt></ruby>みオブジェクトを学ぶことは、留学生のみなさんがプログラミングの「<ruby>初心者<rt>しょしんしゃ</rt></ruby>」から「<ruby>中級<rt>ちゅうきゅう</rt></ruby>者」へステップアップするための大きな<ruby>一歩<rt>いっぽ</rt></ruby>です。

2. forEachやfor-inで<ruby>効率<rt>こうりつ</rt></ruby>よく処理し、
3. StringやMath、Dateで情報を<ruby><rt>賢</rt>かしこ</ruby>く<ruby>加工<rt>かこう</rt></ruby>する。

では、最初の<ruby>組<rt>く</rt></ruby>み<ruby>込<rt>こ</rt></ruby>みオブジェクトとして、「<ruby>配列<rt>はいれつ</rt></ruby>（はいれつ）」について、みてみましょう。
